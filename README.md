# My_projects_by_books_Django_4_in_examples

Этот репозиторий содержит обучающие проекты из книги **"Django 4 в примерах"**.  
Здесь собраны примеры различных веб-приложений на Django 4, включая блог, образовательный сайт, социальную сеть и интернет-магазин.

## Проекты:
- `blog_site` – блог-платформа
- `educational_website` – образовательный сайт
- `social_media_site` – социальная сеть
- `store_shop` – интернет-магазин

Каждый проект находится в отдельной папке.
